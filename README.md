> Production-grade 3D classification system built at 3d Smart Factory.

```
Input Point Cloud (1024 pts)
    ↓
Dynamic Edge Conv Layers (k-NN graph)
    ↓
Feature Aggregation (EdgeConv × 4)
    ↓
Global Max Pooling → MLP Classifier
    ↓
Output: Object Class (10 categories — ModelNet10)
```
